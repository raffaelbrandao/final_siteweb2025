## ACEF - Associação Criança e Família (Salvador, BA)
Este repositório contém o código-fonte do site institucional da ACEF, localizada em Salvador, Bahia. O projeto foi desenvolvido com uma interface focada no apoio à infância e às famílias da comunidade de Rio Sena.

# Link de Visualização
O site está disponível no endereço: https://acef-website-restora-9hos.bolt.host/

# Gestão de Fotos e Imagens
Pasta Local: As fotografias originais estão arquivadas na pasta /fotos deste repositório.

# Armazenamento de Exibição: Atualmente, o site utiliza links hospedados no serviço Catbox para carregar as imagens.

# Edição: Os links de imagem presentes no código podem ser editados para apontar para novos endereços externos ou para os arquivos locais da pasta /fotos.

Ficha Técnica
Tecnologias: React, Tailwind CSS, Lucide Icons.

Idiomas: Português, Francês, Inglês e Espanhol.

Estrutura: Home, Quem Somos, Atividades, Doações e Contato.

## EN FRANCAIS : ACEF - Association Enfant et Famille (Salvador, BA)
Ce dépôt contient le code source du site institutionnel de l'ACEF, située à Salvador de Bahia, Brésil. Le projet est conçu pour présenter les actions de l'association auprès des enfants et des familles du quartier Rio Sena.

Lien de Visualisation
Le site est consultable en direct à l'adresse suivante : https://acef-website-restora-9hos.bolt.host/

Gestion des Photos et Images
Dossier Local : Les photographies originales sont regroupées dans le dossier /fotos de ce dépôt.

Hébergement : Pour l'affichage actuel, le site fait appel à des liens externes stockés sur Catbox.

Modifications : Les URLs des images dans le code peuvent être modifiées pour utiliser de nouveaux liens ou pointer vers les fichiers locaux du dossier /fotos.

Fiche Technique
Stack : React, Tailwind CSS, Lucide Icons.

Langues : Portugais, Français, Anglais et Espagnol.

Pages : Accueil, Qui Sommes-nous, Activités, Dons et Contact.
